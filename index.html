MOST POPULAR PROGRAMMING LANGUAGES

(excluding html/css)

Java

First appearance: May 23, 1995

Type system: Strong, Nominative

Usage:
Android Applications
GUI Applications
Web Applications & Servers

Exemplery code: 


String a = "Hello, world!";
int b = 1;

if (b == 1){		
    System.out.println(a);
}

JavaScript

First appearance: December 4, 1995

Type system: Weak, Dynamic, Duck

Usage:
Web Pages
Web & Mobile Applications
Browser Games

Exemplery code: 


let a = "Hello, world!";
let b = 1;

if(b === 1){
    console.log(a);
}

Python

First appearance: 20 February, 1991

Type system: Strong, Dynamic, Duck

Usage:
AI & Machine Learning
Game Development
Desktop & Mobile Applications

Exemplery code: 


a = "Hello, world!"
b = 1

if b == 1:
    print(a)

C# (C++++)

First appearance: 2000

Type system: Strong, Dynamic, Duck

Usage:
Desktop & Microsoft Applications
Game Development (Mostly Unity)
Web Applications

Exemplery code: 


string a = "Hello, world!";
int b = 1;

if(b == 1){
    Console.Write(a);
}
